Photoshop-like Color Picker for MooTools

This is basically John Dyer's Photoshop-like color picker, but translated from Prototype to MooTools, and with a few added features.

Features

  • Works with MooTools (tested with 1.2);
  • DynamicColorPicker creates multiple instances of the color picker HTML table for you;
  • In fact, it allows you to turn all input fields with a specific class into color-picker fields in one line of JavaScript: 
        window.addEvent('domready', function() {
        DynamicColorPicker.auto(".color-field");
    });
  • When used with MooTools More, autoloads the required JS files so you only have to include one (DynamicColorPicker.js).
